The Aim

This project's goal is to essentially demystify the Git version control system. While many of us are familiar with the git commands. If you ever wondered about the underlying process, you're at the right place.

  • The Locations Model Tutorial

    A short tutorial on the Git Content Model. Learn Git commands in the context of where the code changes happen.

  • Collaboration Scenarios

    Even if one is comfortable with the commands, many. Test yourself with three common situations that arise when working with Git in teams.

NOTE: This is not meant to be a complete and comprehensive guide. There are many awesome resources online for more information.
